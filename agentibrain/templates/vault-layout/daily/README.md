# Daily Logs

Append-only chronological log. One file per day: `YYYY-MM-DD.md`.

## Rules
- **Append-only** — never modify past entries
- Short bullets, raw thoughts, quick captures
- Link to richer notes in `left/`, `right/`, `bridge/` as you go
- The tick-engine may also write summary entries here

## Frontmatter
```yaml
---
date: YYYY-MM-DD
tags: [daily]
status: active
hemisphere: bridge
type: daily
---
```
