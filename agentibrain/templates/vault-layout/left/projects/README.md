# Projects — MUBS Instances

Each technical project is a directory here containing a [MUBS](../../templates/mubs/) (Minimal Unit of Brain Storage).

Start with:

```bash
mkdir left/projects/<your-project>
cp -r templates/mubs/* left/projects/<your-project>/
```

A project can start with just `VISION.md` — the other MUBS files fill in as the work progresses.
