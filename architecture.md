# Architecture

Internals — services, data flows, and the design choices that shape the kernel.

**Architecture** is the wide-angle view of services, data plane, and control plane. **Clusters** describes the arc primitive (the shared memory unit). **Keeper** is the brain-keeper agent that curates the vault. **Telemetry** explains the OTel + ClickHouse + Langfuse pipeline. **Maturity** is the rubric this kernel is graded against.
